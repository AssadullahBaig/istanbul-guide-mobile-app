---

## Project Overview

The Istanbul Smart Tourism Guide is a mobile application designed to help users explore the historical and cultural landmarks of Istanbul through an interactive map interface.

The system integrates location-based services, historical datasets, and interactive visualizations to provide an engaging exploration experience.

Key features include:

- Interactive map-based landmark exploration
- Location-aware landmark discovery
- Category-based landmark filtering
- Search functionality
- Nearby landmark recommendations
